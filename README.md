# Трекер личных финансов

Простое веб-приложение для учета личных доходов и расходов.
Позволяет добавлять операции, фильтровать их по типу и сохранять данные в localStorage.

## Стек
- JavaScript (ES6+)
- React
- TypeScript
- npm
- Vite
- CSS3 (Flexbox)

## Функциональность
- Добавление доходов и расходов
- Переключение типа операций (доходы / расходы)
- Подсчет баланса пользователя в целом
- Подсчет суммы доходов и суммы расходов
- Сохранение данных в localStorage
- Удаление операций по одной или всех сразу

## Запуск проекта

npm install  
npm run dev
